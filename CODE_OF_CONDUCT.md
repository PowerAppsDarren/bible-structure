# Code of Conduct

## Welcome

This project exists to help people study the Bible. Everyone who participates — whether using this template for personal study or contributing back to the shared resource — is welcome here.

## Expected Behavior

- **Respect** — Treat others the way you would want to be treated. Disagree graciously.
- **Good faith** — Assume the best intentions. Approach discussions with humility and a willingness to learn.
- **Constructive contributions** — Focus on helping others grow in understanding. Share insights, ask honest questions, and offer encouragement.
- **Stay on topic** — Keep discussions relevant to Bible study and the purpose of this repository.

## Unacceptable Behavior

- Personal attacks, insults, or harassment of any kind
- Disrespectful or dismissive language toward others or their beliefs
- Spam, self-promotion, or off-topic content
- Deliberate disruption of discussions or contributions

## Reporting Issues

If you experience or witness behavior that violates this code of conduct, please open an issue or contact the repository maintainer directly. All reports will be reviewed and handled with discretion.

## Summary

Study with integrity. Engage with grace. Build each other up.
