## üìö Lessons Learned

- **Windows user PATH ‚âÝ Git Bash PATH.** Claude Code on Windows runs hooks under `/usr/bin/bash` (Git Bash). Bash inherits its own PATH and never picks up `%USERPROFILE%\.local\bin`. **Always use absolute paths in hook commands** ‚Äî bare `mempalace` will silently break.
- For MCP servers in `.mcp.json` on Windows, prefer the full Windows `.exe` path (with escaped backslashes in JSON) over a bare command name. Same PATH-visibility issue applies to whatever spawns the MCP process.
- `mempalace init` is interactive by default. Use `--yes --auto-mine --no-llm` for non-interactive setup, especially in CI or long-running tool calls.
- The cp1252 unicode-arrow crash is cosmetic (only affects `--help`). Set `PYTHONIOENCODING=utf-8` once in your shell profile if it's annoying.
- Mempalace skipped 1,194 of 1,356 files as "already filed" on the second mine run ‚Äî it's content-addressed and idempotent. Safe to re-run anytime.
- `.gitignore` was auto-extended to exclude `mempalace.yaml` and `entities.json`. Per repo conventions, don't `git add -A`; review before committing.
